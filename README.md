# آکادمی تک - وبسایت آموزشی کامپیوتر و شبکه

## درباره پروژه

این وبسایت یک پلتفرم آموزشی کامل برای فروش فیلم‌های آموزشی، مقالات و فایل‌های PDF در حوزه کامپیوتر و شبکه است. سایت با رنگ‌بندی آبی فیروزه‌ای و زرد قهوه‌ای طراحی شده و دارای امکانات کاملی برای فروش آنلاین محتوای آموزشی است.

## ویژگی‌های اصلی

### 🎯 صفحه اصلی
- **هدر زیبا** با لوگو، منوی ناوبری و دکمه‌های ورود/ثبت نام
- **اسلایدشو** تعاملی با کنترل‌های کاربری
- **بخش محصولات** با تب‌های مختلف:
  - پرفروش‌ترین محصولات
  - جدیدترین محصولات  
  - محصولات در حال انجام
  - محصولات تمام شده
- **سایدبار راست** شامل:
  - دسته‌بندی موضوعی
  - آرشیو زمانی
  - لینک‌های مفید
  - محبوب‌ترین مقالات

### 📱 ریسپانسیو و موبایل فرندلی
- طراحی کاملاً واکنش‌گرا
- منوی موبایل با انیمیشن
- بهینه‌سازی برای تمام سایزهای صفحه

### 🛒 سیستم فروش
- سبد خرید تعاملی
- نمایش قیمت و امتیازدهی
- دکمه‌های افزودن به سبد با انیمیشن
- نوتیفیکیشن‌های زیبا

### 🔐 سیستم احراز هویت
- مدال ورود به سیستم
- مدال ثبت نام
- اعتبارسنجی فرم‌ها
- پیام‌های خطا و موفقیت

### 🎨 طراحی و UI/UX
- رنگ‌بندی آبی فیروزه‌ای (#17a2b8) و زرد قهوه‌ای (#ffc107)
- فونت فارسی Vazir
- انیمیشن‌های روان و جذاب
- آیکون‌های Font Awesome
- سایه‌ها و گرادیان‌های مدرن

## ساختار CSS

### 🎨 متغیرهای رنگی
```css
:root {
    --primary-color: #17a2b8;    /* آبی فیروزه‌ای */
    --secondary-color: #ffc107;  /* زرد قهوه‌ای */
    --dark-color: #343a40;       /* خاکستری تیره */
    --light-color: #f8f9fa;      /* خاکستری روشن */
}
```

### 📐 Grid Layout
- Grid اصلی: محتوا + سایدبار
- Grid محصولات: responsive با حداقل 300px
- Grid فوتر: 4 ستون قابل تطبیق

### 📱 Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px  
- Mobile: تا 767px
- Small Mobile: تا 480px
