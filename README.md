# آموزشگاه کامپیوتر و شبکه

یک وبسایت آموزشی کامل برای فروش دوره‌های ویدئویی، کتاب‌های الکترونیکی و مقالات تخصصی در زمینه کامپیوتر و شبکه.

## ویژگی‌ها

### صفحه اصلی
- **هدر زیبا** با اطلاعات تماس و لینک‌های شبکه‌های اجتماعی
- **نوار ناوبری** کامل با منوهای دسته‌بندی شده
- **اسلایدشو** تعاملی با تصاویر و محتوای جذاب
- **بخش محصولات** شامل:
  - محصولات ویژه
  - پرفروش‌ترین محصولات
  - تازه‌ترین محصولات
  - محصولات در حال انجام
  - محصولات تکمیل شده

### صفحات اصلی
- **صفحه اصلی** (index.html) - نمایش کلی سایت
- **بلاگ** (blog.html) - مقالات و آموزش‌ها
- **درباره ما** (about.html) - معرفی مدرس و آموزشگاه
- **فروشگاه** (shop.html) - نمایش و فروش محصولات
- **تماس با ما** (contact.html) - فرم تماس و اطلاعات

### سایدبار راست
- جستجو در محصولات
- دسته‌بندی موضوعی
- آرشیو تاریخی
- لینک‌های مفید
- خبرنامه

### فوتر
- دسته‌بندی محصولات
- لینک‌های صفحات
- شبکه‌های اجتماعی
- اطلاعات تماس
- معرفی مالک سایت

### ویژگی‌های فنی
- **طراحی ریسپانسیو** با Bootstrap 5
- **رنگ‌بندی** آبی فیروزه‌ای و زرد قهوه‌ای
- **انیمیشن‌ها** و افکت‌های زیبا
- **مدال‌های** ورود و ثبت نام
- **سبد خرید** کاملاً عملکردی
- **فیلترینگ** و مرتب‌سازی محصولات

## فناوری‌های استفاده شده

- **HTML5** - ساختار صفحات
- **CSS3** - طراحی و استایل‌دهی
- **JavaScript** - تعامل و عملکرد
- **Bootstrap 5** - فریمورک CSS
- **Font Awesome** - آیکون‌ها
- **Google Fonts** - فونت فارسی وزیر

## نحوه استفاده

### راه‌اندازی
1. فایل‌ها را در یک پوشه قرار دهید
2. فایل `index.html` را در مرورگر باز کنید
3. از لینک‌های موجود برای ناوبری استفاده کنید

### ساختار فایل‌ها
```
/
├── index.html          # صفحه اصلی
├── blog.html           # صفحه بلاگ
├── about.html          # صفحه درباره ما
├── shop.html           # صفحه فروشگاه
├── contact.html        # صفحه تماس
├── style.css           # استایل‌های CSS
├── script.js           # کدهای JavaScript
└── README.md           # این فایل
```
